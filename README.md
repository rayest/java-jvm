### JAVA 字节码
``` shell
1. 编写 Demo_01.java 示例
2. 编译：javac Demo_01.java --> Demo_01.class
3. 查看和编辑十六进制文件：vim -b Demo_01.class --> :%!xxd
4. 重新查看二进制文件：:%!xxd -r
5. 查看编译后的字节码文件：javap -verbose -p Demo_01.class
```
